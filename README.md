Mysterium Animæ — Modelo de la Psique de C. G. Jung
Visualización interactiva del modelo psicológico de Carl Gustav Jung (1875–1961) con estética alquímico-editorial.
Contenido
El artefacto está estructurado en cuatro secciones navegables:
Mandala del Alma — las tres capas concéntricas de la psique (conciencia, inconsciente personal, inconsciente colectivo) con nodos interactivos para Persona, Ego, Sombra, Ánima/Ánimus y Self.
Tipos Psicológicos — la brújula funcional de ocho tipos: dos actitudes (introversión/extroversión) × cuatro funciones (pensamiento/sentimiento/sensación/intuición).
Arquetipos — diez arquetipos principales del inconsciente colectivo con descripción, manifestaciones culturales y etimología.
Opus Alchymicum — las tres fases del proceso de individuación: Nigredo · Albedo · Rubedo.
Tecnología
HTML5 + CSS3 + JavaScript vanilla (sin dependencias).
SVG nativo para mandala y brújula tipológica.
Tipografías: Cormorant Garamond, EB Garamond y Cinzel (Google Fonts).
Diseño responsive.
